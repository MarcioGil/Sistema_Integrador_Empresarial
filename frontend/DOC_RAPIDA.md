# Documentação Rápida – Sistema Integrador SaaS

## 1. Login
- Acesse o sistema e faça login com seu usuário e senha.
- Se não tiver cadastro, solicite ao administrador.

## 2. Assinatura
- Veja seu plano e vencimento no painel.
- Clique em “Assine agora” para pagar via Stripe ou Mercado Pago.
- Após pagamento, o sistema libera automaticamente.

## 3. Funcionalidades
- Gerencie clientes, vendas, estoque, financeiro, fornecedores e usuários.
- Use o app no celular, instale como PWA ou Android.

- Sempre que houver nova versão, atualize o app ou instale novamente.

## 6. Segurança
- Todos os dados são protegidos por autenticação JWT e HTTPS.
